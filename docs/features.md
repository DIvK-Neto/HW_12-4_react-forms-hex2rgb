# Функциональность конвертера HEX → RGB

## Основные возможности

- Поле ввода принимает любые символы. Ожидаемый формат: `#RRGGBB` (7 символов, регистр не важен).
- Начальное значение: `#9921ff`.
- **Индикатор оставшихся символов**:
  - Если в поле меньше 7 символов, под полем ввода отображается сообщение: `Осталось ввести: X символов`, где X = 7 – текущая длина.
  - Сообщение учитывает склонение: «1 символ», «2 символа», «5 символов».
  - Индикатор работает для любой длины (от 0 до 6), даже если пользователь удалил решётку.
- **Проверка и преобразование**:
  - Когда в поле становится **ровно 7 символов**, происходит проверка HEX.
  - Корректный HEX (начинается с `#`, затем 6 шестнадцатеричных цифр) → отображается RGB, фон страницы меняется на указанный цвет.
  - Некорректный HEX → блок результата становится красным, выводится `Ошибка! Неверный HEX`, фон страницы не меняется.
- **Обработка длинного ввода**:
  - Если в поле **более 7 символов** (например, 15), это считается ошибкой: блок результата красный, сообщение `Ошибка! Неверный HEX`, фон не меняется.
- При размонтировании компонента фон страницы сбрасывается.

## Примеры работы

| Ввод (длина)              | Отображение в нижнем блоке          | Фон страницы                                  |
| ------------------------- | ----------------------------------- | --------------------------------------------- |
| `#` (1)                   | Осталось ввести: 6 символов         | не меняется (последний корректный или пустой) |
| `#9` (2)                  | Осталось ввести: 5 символов         | не меняется                                   |
| `#9921f` (6)              | Осталось ввести: 1 символ           | не меняется                                   |
| `9921ff` (6, без `#`)     | Осталось ввести: 1 символ           | не меняется                                   |
| `#9921ff` (7, корректный) | `rgb(153, 33, 255)`                 | фиолетовый                                    |
| `#00FFAA` (7, корректный) | `rgb(0, 255, 170)`                  | зелёный                                       |
| `#GGGGGG` (7, неверный)   | Ошибка! Неверный HEX (красный блок) | остаётся прежним                              |
| `#1234567` (8)            | Ошибка! Неверный HEX (красный блок) | остаётся прежним                              |
| `#1234567890` (11)        | Ошибка! Неверный HEX (красный блок) | остаётся прежним                              |

## Особенности реализации

- Утилита `hexToRgbString` (файл `src/utils/hexToRgb.ts`) выполняет валидацию и преобразование.
- Компонент `ColorConverter` управляет состояниями, индикатором и фоном.
- Используется React + TypeScript, изменение фона через `document.body.style.backgroundColor`.
- При ошибке или длине >7 фон страницы не меняется, остаётся последний корректный цвет (если был).
