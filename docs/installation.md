# Установка и запуск

## Требования

- Node.js 20 или выше
- npm (устанавливается вместе с Node.js)

## Инструкция

1. Клонировать репозиторий:

```bash
git clone https://github.com/DIvK-Neto/HW_12-4_react-forms-hex2rgb.git
cd HW_12-4_react-forms-hex2rgb
```

2. Установить зависимости:

```bash
npm install
```

1. Запустить в режиме разработки:

```bash
npm run dev
```

4. Открыть в браузере адрес `http://localhost:5173`.

## Сборка для продакшена

```bash
npm run build
```

Собранные файлы будут находиться в папке `dist/`.

## Проверка кода

- Линтинг: `npm run lint`
- Форматирование: `npm run format`
