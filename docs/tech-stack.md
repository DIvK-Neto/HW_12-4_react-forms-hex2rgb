# Используемые технологии

## Основной стек

- **React** (версия 18) – библиотека для построения пользовательского интерфейса.
- **TypeScript** – типизация JavaScript, повышает надёжность кода.
- **Vite** – сборщик и инструмент разработки (быстрая горячая перезагрузка).

## Инструменты качества кода

- **ESLint** (flat config, `eslint.config.mjs`) – статический анализ кода, проверка синтаксиса и стиля.
- **Prettier** – автоматическое форматирование кода (единый стиль).

## Деплой

- **GitHub Pages** – хостинг статики.
- **GitHub Actions** – CI/CD для автоматической сборки и публикации.

## Дополнительно

- **CSS** – стилизация компонента (без дополнительных фреймворков).

## Требования к окружению

- Node.js 20 LTS или выше
- npm (входит в состав Node.js)
